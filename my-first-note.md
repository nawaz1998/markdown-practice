## Escaping Characters

This is italic: *italic*

This is not italic: \*italic\*

This is bold: **bold**

This is not bold: \*\*bold\*\*

This is a header: # Header

This is not a header: \# Header

This is a blockquote: > Quote

This is not a blockquote: \> Quote
